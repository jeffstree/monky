## Description
This website allows you to play three different daily Wordle games, with valid words based on information about Pokemon, cats, and birds. This information is obtained using PokeAPI, The Cat API, and NuthatchAPI, respectively. Logging in allows you to keep track of your data, including wins and streaks.
